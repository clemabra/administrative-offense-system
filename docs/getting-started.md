# Getting Started with OPAS

## Prerequisites
- Node.js 16+
- .NET 8.0 SDK
- Git
- IDE (VS Code recommended)

## Quick Setup

### 1. Clone Repository
```bash
git clone [repository-url]
cd opas
```

### 2. Backend Setup
```bash
cd backend
dotnet restore
dotnet run
```
The backend will start at http://localhost:5000

### 3. Frontend Setup
```bash
cd frontend
npm install
npm run serve
```
The frontend will start at http://localhost:8080
